19 This class template provides a mechanism to save a stream's I/O formatting
20 properties and, at a later date, restore them to those that were previously
21 saved (both explicitly on request and via RAII on object destruction).
22 
23 A simple example:
24 
25 void report_hex( unsigned const n )
26 {
27  awo::savefmt const saver{ std::cout };
28 
29  std::cout << std::hex << std::uppercase << n << std::endl;
30 }
31 
32 On return from the function, the saver object's destructor will
33 return std::cout's formatting parameters back to how they were.
34 
35 Another example:
36 
37 void f()
38 {
39  std::cout << std::dec << 200 << std::endl;
40  std::cout << awo::savefmt{} << std::hex << 200 << std::endl;
41  std::cout << 200 << std::endl;
42 }
43 
44 The second reporting line changes the radix before formatting the
45 value but, at the end of the enclosing expression, the temporary
46 savefmt's destructor will restore the formatting parameters that
47 were in effect in the stream before the saver was created.
48 
49 A variety of other, more complex, scenarios can be dealt with
50 using the default constructor, capture(), restore(), release(),
51 the move-constructor, move-assignment operator and operator bool.
52 
53 Since the introduction of rvalue-references to the C++ language,
54 this implementation has adopted their use, thereby circumventing
55 the dodgy-looking const-casting that helped facilitate the origin
56 formatted extraction/insertion (>>, <<) operators.
57 */
